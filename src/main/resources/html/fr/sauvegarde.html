Sauvegarder ou restaurer les donées

Il est vivement recommendé de faire des sauvegardes régulières. Les sauvegardes permettent de restaurer vos donnés quand (par exemple) vous perdez ou bloquez votre clé. Les fichiers de sauvegarde ne peuvent être utilisés que sur un token vide. Lorsque l'entrée du menu 'Sauvegarde' est sélectionnée, deux réactions sont possibles (la différence se voit sur le bouton de validation) :

  • Si au minimum un Groupe est présent sur le token, la fenêtre 'Sauvegarder' est ouverte.
  • Si aucun Groupe n'existe sur le token, la fenêtre 'Restaurer' est ouverte.

Sauvegarder les données

Sélectionez l'endroit où enregistrer la sauvegarde et entrez le mot de passe qui protègera le fichier par chifrement. Après avoir cliqué sur le bouton 'Sauvegarder le token', une fenêtre affichant la progression de la sauvegarde est affichée. Si une erreur arrive lors de la sauvegarde, elle sera affichée dans la barre de progression et la fenêtre ne se fermera pas automatiquement. Dans l'autre cas, la fenêtre de progression se fermera automatiquement à la fin de la sauvegarde.

Restaurer les données

Sélectionez l'endroit où lire la sauvegarde et entrez le mot de passe qui déverrouillera le fichier. Après avoir cliqué sur le bouton 'Restaurer le token', une fenêtre affichant la progression de la restauration est affichée. Si une erreur arrive lors du processus, elle sera affichée dans la barre de progression et la fenêtre ne se fermera pas automatiquement. Dans l'autre cas, la fenêtre de progression se fermera automatiquement à la fin de la restauration.
