Première utilisation

Lors de la première utilisation du token ou après une mise à jour du token, l'utilisateur doit mettre à jour le mot de passe. En effet, le mot de passe par défaut est 'password'. Pour cela, il faut commencer par se connecter (voir Connexion) avec le mot de passe par défaut. Il faut ensuite modifier le mot de passe en sélectionnant l'entrée du menu 'Fichier' → 'Changer le mot de passe'.

Une icone est affichée à la droite de chaque champ, indiquant si les données renseignées sont correctes ou non. Une info-bulle est également présente en cas d'erreur, permettant de conseiller l'utilisateur. Voici la liste des vérifications réalisées dans cette boite de dialogue :

  • Le mot de passe ne peut pas être vide
  • Les mots de passes doivent être identiques
  • Le mot de passe doit contenir au moins 8 caractères

Il est conseillé de modifier régulièrement le mot de passe. Pour une sécurité optimale, il est recommandé d'utiliser un mot de passe d'une taille minimale de 16 caractères, en utilisant des majuscules, des minuscules, des chiffres ainsi que des caractères spéciaux.
