Préférences

L'utilisateur peut configurer certains paramètres. Parmi les option disponibles, on trouve :

  • Langue (pour que tous les éléments soient traduits, un redémarrage de l'application est nécessaire).
  • Thème de l'interface utilisateur (Classique ou Sombre).
  • L'ensemble des caractères spéciaux par défaut utilisé dans le générateur de mot de passe (voir Créer une Entrée).
  • Timer de déconnexion (la valeur est donnée en secondes). Voir Connexion pour plus de détails.
