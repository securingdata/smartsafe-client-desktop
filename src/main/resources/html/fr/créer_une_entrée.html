Créer une Entrée

Cette action est accessible au travers l'entré,e du menu 'Fichier' → 'Nouvelle Entrée'. Un Goupe doit être sélectionné pour pouvoir réaliser cette action.

L'Entrée crée sera ajoutée au Groupe sélectionné. Seulement deux champs doivent être reseignés : 'Identifiant' et 'Nom d'utilisateur'. Les autres champs sont optionnels.

La date d'expiration est usilisée pour les besoins d'affichage de l'Entrée mais n'en bloque pas son utilisation lorsqu'elle est dépassée. L'URL est utilisée pour la navigation rapide et les Notes sont disponibles pour sauvegarder une information associée à une Entrée.

Le mot de passe peut-être renseigné par l'utilisateur mais un générateur de mots de passe est disponible en cliquant sur le bouton 'Aléatoire'. La taille du mot de passe et les cractères générés peuvent être configurés.
