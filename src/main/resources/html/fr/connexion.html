Connexion

Sélectionner le lecteur qui contient le token SmartSafe et entrer le mot de passe qui a été personnalisé lors de l'initialisation.

Quand le mot de passe est erronné, le compteur d'essais est décrémenté. Lorsque le compteur atteint 0, toutes les données contenues dans le token sont perdues.

Lorsque le mot de passe est correct, l'accès aux données du token est autorisé. Le lecteur utilisé sera également automatiquement sauvegardé pour la prochaine session.

Dès qu'une session est ouverte, un compte à rebours est lancé. Lorsque ce compte à rebours atteint zéro, le token est automatiquement déconnecté. Voir Préférences pour configurer ce compte à rebours.
