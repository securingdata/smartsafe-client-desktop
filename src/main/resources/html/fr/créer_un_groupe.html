Créer un Groupe

Cette action est disponible en cliquant sur l'entrée du menu 'Fichier' → 'Nouveau Groupe'.

De même que pour créer un dossier, céer un Groupe nécessite de renseigner un nom. Une erreur peut survenir lors de la création d'un Groupe (par exemple s'il ne reste pas assez de mémoire sur le token pour créer le Groupe).

Attention: Pour l'instant, les Groupes ne peuvent pas être renommés mais cette fonctionnalité sera ajoutée lors d'une mise à jour future.
