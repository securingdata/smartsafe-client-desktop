Créer un Groupe

Cette action est disponible en cliquant sur l'entrée du menu 'Fichier' → 'Nouveau Groupe'.

De même que pour créer un dossier, céer un Groupe nécessite de renseigner un nom. Une erreur peut survenir lors de la création d'un Groupe (par exemple s'il ne reste pas assez de mémoire sur le token pour créer le Groupe).

Une icone est affichée à la droite de chaque champ, indiquant si les données renseignées sont correctes ou non. Une info-bulle est également présente en cas d'erreur, permettant de conseiller l'utilisateur. Voici la liste des vérifications réalisées dans cette boite de dialogue :

  • Le nom ne peut pas être vide
  • Un autre groupe a déjà ce nom
  • La taille du nom ne peut pas excéder 30 caractères
