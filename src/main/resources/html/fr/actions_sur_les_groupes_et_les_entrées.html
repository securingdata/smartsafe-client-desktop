Actions sur les Groupes et les Entrées

Lorsqu'une Entrée est sélectionnée, plusieurs actions peuvent être réalisées, en utilisant l'entrée du menu 'Edition' :

  • l'Entrée peut-être éditée,
  • l'URL fournie peut être ouverte dans le navigateur web par défaut,
  • le nom d'utilisateur ou le mot de passe peut-être copié dans le presse-papier,
  • enfin, le mot de passe peut-être montré à l'utilisateur.
Dans le dernier cas, pour des raisons de sécurité, sélectionner une autre Entrée re-cachera le mot de passe précédemment montré.

Lorsque l'utilisateur déclenche l'action 'Supprimer', deux réactions sont possibles :

  • Si aucune Entrée n'est sélectionnée, la seule action disponible est de supprimer le Groupe sélectionné.
  • Si une Entrée est sélectionnée, l'utilisateur peut choisir entre supprimer l'Entrée sélectionnée ou supprimer le Groupe sélectionné.
